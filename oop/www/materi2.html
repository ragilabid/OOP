    Pembungkusan data / Enkapsulasi

    Enkapsulasi merupakan teknik yang membuat variabel menjadi bersifat personal dan menyediakan akses ke variabel melalui public method. Jika variabel dideklarasikan sebagai private, maka variabel tidak bisa diakses oleh siapapun di luar class, dengan demikian variabel disembunyikan di dalam class.

    Modifier

    Terdapat 4 macam access modifiers di JAVA, yaitu: public, private, protected dan default. Tiga tipe akses pertama tertulis secara ekplisit pada kode program untuk mengindikasikan tipe akses, sedangkan yang keempat merupakan tipe default yang tidak memerlukan penulisan keyword atas tipe.

    • add
      Public

      Hak akses public dapat diakses oleh sembarang objek manapun dan di manapun posisinya serta dengan apapun caranya.

    • add
      Protected

      Suatu data maupun method yang dideklarasikan dengan tingkat akses protected dapat diakses oleh class yang memilikinya dan juga oleh class – class yang masih memiliki hubungan turunan.

    • add
      Default

      Untuk hak akses default ini, sebenarnya hanya ditujukan untuk class yang ada dalam satu paket, atau istilahnya hak akses yang berlaku

    • add
      Private

      Dengan mendeklarasikan data dan method menggunakan tingkat akses private, maka data dan method tersebut hanya dapat diakses oleh class yang memilikinya.

    Tipe Akses Data

    Contoh penggunaan akses data ditunjukkan pada baris kode di bawah ini,

    class atas { -> //default
    public int a; -> //public
    protected int b; -> //protected
    private int c; -> //private
    }
    class bawah { -> //default
    public static void main(string[]args){
    atas objek = new atas();
    objek.a = 2;
    objek.b = 3;

    System.out.println("nilai a:"+objek.a);
    System.out.println("nilai b:"+objek.b);
    }
    }

    Penjelasan

    program diatas terdiri dari dua class yaitu class sekunder yang berisi variabel a, b dan c dengan tingkat akses yang berbeda, dan class primer yang berisi objek untuk melakukan instance pada class turunan, objek pada class primer hanya dapat mengisi nilai pada variabel a dan b karena kedua variabel tersebut memiliki tingkat akses public dan protected, karena variabel c memiliki tingkat akses private maka objek pada class primer tidak bisa mengisi variabel tersebut.
