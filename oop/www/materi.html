    Paradigma PBO

    Ide dasar pada pemrograman berorientasi objek (POB) adalah mengkombinasikan data dan fungsi untuk mengakses data menjadi sebuah kesatuan unit yang dikenal sebagai objek.

    Analogi Objek

    Analogi dengan hal ini, kalau seseorang bermaksud menggunakan objek, ia cukup mengirim suatu pesan ke objek dan objek itu sendiri yang akan menanganinya. Sebuah gambaran tentang objek yang berisi data dan fungsi yang memanipulasi data dapat dilihat pada gambar 1

    Paradigma bahasa pemrograman memberikan model untuk programmer dalam menulis listing program. Paradigma perbedaan dalam bahasa pemrograman sebagai berikut:

      • Program ini dibagi menjadi modul dan modul tersebut kemudian dibagi menjadi fungsi.

      • Fungsi satu objek dapat mengakses fungsi objek lain.

    Instalasi Java SDK & Netbeans

    Instalasi Java SDK

    LANGKAH 1

    Buka file instalasi program setup java seperti pada gambar

    LANGKAH 2

    Kemudian muncul kotak dialog seperti gambar disamping. Klik “next” untuk melanjutkan

    LANGKAH 3

    Perhatikan folder instalasi java Tekan “change” untuk mengubah lokasi tempat penyimpanan (optional), tekan next untuk melanjutkan.

    LANGKAH 4

    Tunggulah hingga proses instalasi JDK selesai

    LANGKAH 5

    JDK telah selesai di instal, Program akan secara otomatis melakukan instalasi JRE. Tekan next untuk melanjutkan

    LANGKAH 6

    Tunggulah hingga proses instalasi JRE selesai

    LANGKAH 7

    Proses instalasi JRE telah selesai, tekan close untuk mengakhirinya

    LANGKAH 8

    Buka dialog System Properties melalui kotak dialog control panel. Klik Advanced System Settings

    LANGKAH 9

    Pada kotak dialog System Properties klik tab Advanced. Kemudian klik Environtment variables

    LANGKAH 10

    Temukan variabel Path pada system variables. Kemudian klik edit

    LANGKAH 11

    Tambahkan nilai baru pada variable value yaitu folder binary file JDK, yaitu C:\Program Files\Java\jdk1.8.0_51\bin

    Instalasi Netbeans

    LANGKAH 1

    Klik 2 kali pada installer NetBeans.

    LANGKAH 2

    Centang “I accept.....” Lalu klik next.

    LANGKAH 3

    Lalu pilih tempat penyimpanan instalasi (biarkan saja). Klik Next, tunggu proses instalasi selesai. Lalu klik finish.
